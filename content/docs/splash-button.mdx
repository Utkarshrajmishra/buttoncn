## Installation

```npm
npx shadcn@latest add http://localhost/300/r/splash-button.josn
```


## Usage

```tsx title="usage.tsx"
import { SplashButton } from "../buttoncn/splash-button";

const SplashButtonPreview = () => {
    return (
        <div>
            <SplashButton >
                Hover Me
            </SplashButton>
        </div>
    );
};

export default SplashButtonPreview;
```

## API Reference

### Splash Button

The **Splash Button** component is a wrapper around the **button** element that adds a variety of styles and functionality.
